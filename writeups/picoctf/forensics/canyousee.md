# picoCTF - CanYouSee

## 問題概要
- ジャンル：Forensics
- 難易度：Easy
- 問題の要点：
  一見すると通常の画像ファイルだが、画像の「見た目」ではなく、
  画像に付随する情報（メタデータ）に注目する必要がある問題。

## 解法
- ヒントに「How can you view the information about the picture?」とあり、
  画像そのものではなく「画像の情報」を確認する必要があると判断した
- zipファイルを展開すると、画像ファイル（jpg）が1つだけ含まれていた
- 画像を通常表示してもflagは確認できなかったため、
  ファイル形式や構造が想定通りかを疑う方針に切り替えた
- PowerShellを用いて画像ファイルをバイト列として読み込み、
  ASCII文字列として扱うことで、中に文字情報が含まれていないかを調査した
- その結果、画像内にXMP（XML形式のメタデータ）が埋め込まれていることを確認した
- XMP部分を切り出すと、Base64形式の文字列が含まれていた
- Base64文字列をデコードし、flagを取得した

## 学んだこと
- 画像ファイルは「見た目」だけでなく、メタデータを持つことがある
- XMPやEXIFといったメタデータは、画像を開いただけでは確認できない
- ヒントに「information」や「form」が出た場合、
  メタデータやファイル構造を疑う初動は有効である
- バイナリ全体を読むのではなく、必要な範囲だけを切り出すことで、
  環境差や処理の重さを回避できる

## 次に活かす視点
- 画像が出てきた場合、まず「見た目」ではなく「情報（メタデータ）」を疑う
- ヒントに information / metadata / form などがあれば、
  XMP・EXIF・strings などの確認を優先する
- 重い処理は避け、先頭や特定範囲のみを調べる癖をつける
