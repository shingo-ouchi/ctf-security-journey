# picoCTF - Inspect HTML

## 問題概要
- ジャンル：Web Exploitation
- 難易度：Easy
- 画面には表示されていないHTMLコメント内に、flagが含まれている問題

## 解法
- 画面表示の情報だけではなく、HTML自体に隠された情報がある可能性を疑った
- 開発者ツール（またはページソース）でHTMLを確認した
- 画面表示には関係ないHTMLコメントを発見し、その中にflagが含まれていることを確認した
- コメント内のflagをそのまま提出した

## 学んだこと
- HTMLコメントはユーザーから見えにくいが、誰でも確認できる情報である
- 「表示されていない＝安全」ではなく、クライアントに送信されている時点で情報は保護されていない

## 次に活かす視点
- まずHTMLコメント（`<!-- -->`）を確認する
- 画面に表示されていない情報がHTML内に含まれていないかを疑う
- 「隠しているだけ」の設計になっていないかを意識する
